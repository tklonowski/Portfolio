Welcome to Tom Klonowski's Portfolio

This website is currently in progress. It not only describes my skills, but it puts some of them to use. It was written in .NET MVC 4, using Razor syntax. The technologies used in this website so far include, but are not limited to:

  • Ninject for Dependency Injection and IoC
  • ServiceStack for a Service framework
    • SwaggerUI to test ServiceStack services
  • Log4Net for logging purposes
  • AngularJS for full MVC JS support
  • Moq for automated mock Unit Testing
  • EntityFramework for data manipulation
    • CodeFirst for database creation/maintenance
  • jQuery for further JS support
