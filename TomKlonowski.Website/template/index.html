Here are most of the technologies used in this site

This website is fully responsive. A mobile first approach is taken in order to deliver a consistant experience across all devices.

Ninject

Ninject helps you use the technique of dependency injection to break your applications into loosely-coupled, highly-cohesive components.

ServiceStack

One framework for all your services and web apps that's intuitive and Easy-to-use!

Swagger UI

Swagger UI is part of the Swagger project. The Swagger project allows you to produce, visualize and consume your OWN RESTful services.

log4net

The Apache log4net library is a tool to help the programmer output log statements to a variety of output targets.

AngularJS

AngularJS is a structural framework for dynamic web apps. It lets you use HTML as your template language and lets you extend HTML's syntax.

Moq

The most popular and friendly mocking framework for .NET
